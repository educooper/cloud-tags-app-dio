## :white_check_mark: Requirements / Requisitos ##

Antes de iniciar :checkered_flag:, você precisa ter instalado o [Git](https://git-scm.com) e .NET Core.

## :checkered_flag: Starting / Início ##

```bash
# Clone este projeto
$ git clone https://github.com/educooper/cloud-tags-app-dio

# Acesse
$ cd mind-cloud-apps-dio

#principais comandos

#instancia o rail ao projeto juntamente com o postgreesql
rails new cloud-tags-app-dio -d postgreesql -T 

Editar o GEMFILE adicionando o DEVISE - uma solução completa de autenticação de usuários
gem 'devise'
bundle install
rails generate devise install

# Execute o projeto
$ rails server run
```
